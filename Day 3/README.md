# Treasure Island project:

## Concepts : Conditional Statements
### Nested if
```

if condition:
       if another condition:
           do this
       else:
           do this
else:
     do this
```

### if / elif/ else:
```
if condition:
   do this
elif another condition:
   do this
else:
   do this
```
